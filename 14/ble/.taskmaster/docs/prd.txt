# ESP32 BLE 강의자료 제작 프로젝트

## 프로젝트 개요
ESP32 Bluetooth Low Energy (BLE) 기술에 대한 포괄적인 강의자료를 제작하는 프로젝트입니다. 기존 README.md 파일을 기반으로 하여 웹 크롤링을 통해 이미지와 추가 정보를 수집하고, 체계적인 강의자료로 재구성합니다.

## 목표
- ESP32 BLE 기술에 대한 완전한 강의자료 작성
- 이론과 실습이 균형잡힌 교육 콘텐츠 제작
- 시각적 자료(이미지, 다이어그램)가 포함된 마크다운 문서
- 500줄 제한을 고려한 모듈화된 문서 구조

## 주요 기능 요구사항

### 1. 웹 크롤링 및 자료 수집
- RandomNerdTutorials ESP32 BLE 가이드에서 이미지 추출
- 코드 예제 및 설명 텍스트 수집
- 이미지 다운로드 또는 URL 추출
- 크롤링 실패 시 대안 방안 수립

### 2. 콘텐츠 구조화
- 기존 README.md 파일 분석
- 논리적 학습 순서에 따른 내용 재구성
- 이론 설명 + 실습 코드 + 시각적 자료 조합

### 3. 문서 작성 및 관리
- 마크다운 형식으로 강의자료 작성
- 500줄 제한 고려한 파일 분할 전략
- 이미지 삽입 및 레이아웃 최적화

## 기술적 제약사항
- Filesystem MCP: 500줄 초과 시 무한로딩 발생
- Playwright MCP: 이미지 크롤링 안정성 문제
- 단일 파일 크기 제한으로 인한 모듈화 필요

## 성공 기준
- 완전한 ESP32 BLE 강의자료 완성
- 모든 이미지가 정상적으로 포함
- 500줄 제한 내 각 섹션 완성
- 학습자가 따라하기 쉬운 구조
